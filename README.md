# Password Strength Analyzer

## Built With
- Python Flask
- HTML
- CSS

## Features
- Checks password length
- Checks complexity
- Detects common passwords
- Gives suggestions for stronger passwords

## How to run
1. Install Flask:
   pip install flask

2. Run project:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/
